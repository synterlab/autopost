# AutoPost

Social media scheduling SaaS web app — built for [autopost.auction](https://autopost.auction)

## Features
- 🗓️ Smart scheduling with AI best-time suggestions
- 📊 Analytics dashboard (reach, engagement, follower growth)
- 🌐 Multi-platform: Instagram, Facebook, LinkedIn, X, TikTok, YouTube, Pinterest, Telegram
- 👥 Team collaboration with approval workflows
- 🔁 Recurring / evergreen content autopilot

## Stack (Single HTML prototype)
- Pure HTML/CSS/JS — zero dependencies
- Dark premium design system (Syne + DM Sans)
- Fully responsive — mobile & desktop

## Pages
- **Landing** — full marketing page (Navbar, Hero, Platforms, Features, How It Works, Stats, Pricing, Testimonials, CTA, Footer)
- **Auth** — login / register with validation & loading states
- **Dashboard** — stats, schedule calendar, analytics, accounts, settings + post composer modal

## Deploy to Vercel
1. Import this repo at [vercel.com/new](https://vercel.com/new)
2. No build settings needed — just click Deploy
3. All routes rewrite to `autopost.html` via `vercel.json`
